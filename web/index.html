Expected format for processes

The file must include a header row with the following columns (case-insensitive, flexible order):

PID, Arrival, Burst, Priority, Pages, Threads (optional), Forks (optional)

Example
PID,Arrival,Burst,Priority,Pages,Threads,Forks
1,0,5,2,3,2;3,
2,1,3,1,2,,1
3,2,8,3,4,3,2;3
4,3,2,4,1,,
5,4,4,2,2,4,
6,6,6,3,3,2;4,3
  • Threads / Forks accept three formats:
  • empty · 0 · -  →  no children
  • 2;3  →  list of bursts separated by ;
  • 3  →  count N (burst split automatically)
  • If the CSV includes threads or forks, toggles will be enabled automatically.
  • Maximum: 4 threads and 3 forks per process.
